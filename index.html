Privacy Policy for Tabman

Data Collection and Usage

Tabman operates entirely within your browser and does not collect, transmit, or store any personal data externally. The extension:

  • Reads tab information (URLs and titles) only to provide search and organization features
  • Stores window names locally in your browser using Chrome's storage API
  • Does not track browsing history
  • Does not send any data to external servers
  • Does not use analytics or tracking tools

Data Storage

All data (including custom window names) is stored locally in your browser using Chrome's built-in storage API and is only accessible by the extension itself.

Permissions

  • The 'tabs' permission is used solely to provide tab management functionality
  • The 'storage' permission is used only to save window names locally in your browser

Changes to This Policy

Any future changes to this privacy policy will be posted here and will be effective when they are posted.
